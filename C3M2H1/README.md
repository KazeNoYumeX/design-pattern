# Shortcut Setting Mechanism 快捷鍵設置機制

## System Requirements

The following are required to function properly.

* PHP Version：PHP 8.2.x

### Composer install

```bash
    composer install
```

### Run Showdown

```bash
     php .\public\Main.php
```

### Run pest test

```bash
     ./vendor/bin/pest
```

### Run Pint

```bash
    ./vendor/bin/pint
```
