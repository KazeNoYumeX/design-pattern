# Prescriber System 處方診斷系統

## System Requirements

The following are required to function properly.

- PHP Version：PHP 8.3.x

### Composer install

```bash
  composer install
```

### Run Showdown

```bash
  php .\public\Main.php
```

### Run pest test

```bash
  ./vendor/bin/pest
```

### Run Pint

```bash
  ./vendor/bin/pint
```
